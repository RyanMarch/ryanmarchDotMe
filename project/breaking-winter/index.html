Breaking Winter

Original Composition

Apr 2014

Breaking Winter is a relaxed song with a hip-hop inspired beat. I wanted to create something different from past works, but was still music that I enjoy. With Breaking Winter, I tried to make a song that was melodic, but did not have an obvious distinct melody. I wanted to include many ideas and synthesizers that I came across during my composition class but wasn't able to utilize in past projects. The song features extensive use of automation, and production techniques that I have picked up in other classes.

Tools Used:
