Forecast Weather

Windows App

Designer & Developer • February 2013

The Product

Forecast Weather is an award-winning Windows 8 app designed to deliver simple 7-day weather forecasts for user-entered locations. Created during a hackathon, the app focused on flat, information-first typography and seamless navigation with the Windows 8 Metro design language.

The flat Metro-style Forecast Weather app icon.

The Hackathon

The application was designed and coded entirely in a single day at a rapid-prototyping workshop held at Capital University. The event was hosted by Brian Prince, Principal Cloud Technical Evangelist at Microsoft (a key player in Azure online services) and a Capital alum.

Building a functional weather app in four hours required extremely rapid design iteration, smart data integration, and ruthless layout simplicity to guarantee a same-day release to the Windows App Store.

What's the absolute simplest weather app? Give a location and receive a forecast. This app is just that.

Recognition

Following its launch, Forecast Weather was selected as an award winner in Microsoft's DreamSpark App Challenge.

The app was eventually retired and removed from the Windows Store in September 2016 following changes to the Windows platform architecture.

Project Specs

Developing in the Windows 8 Touch Develop system required learning specific design constraints and platform paradigms:

  • Platform: Built using Microsoft's Touch Develop platform, and using data from the National Weather Service to fetch live weather data.
  • Interface: Designed for Windows 8 tablets and PCs with large touch targets.
