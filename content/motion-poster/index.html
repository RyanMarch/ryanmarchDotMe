The Product

Motion Poster is an easy-to-use, browser-based digital signage solution designed for fundraisers, galas, and corporate gatherings. It transforms static event displays into dynamic, interactive art that acts as a living, breathing centerpiece to elevate the atmosphere of any physical space.

Built with a "set and forget" philosophy, it allows event hosts to customize text, branding, and layouts in real-time. The display updates inside the browser without requiring developer support, server deploys, or an active internet connection during the event.

Living Centerpiece: The Spring Blossom theme layers organic falling cherry blossom particles behind an elegant typography layout.

Live Editor & Name Management

Adjust settings seamlessly on the fly using a slide-out control drawer. Accessible via keyboard shortcuts or a discreet corner tap, the editor empowers hosts to calibrate the display in real-time without disrupting the viewer experience:

  • Drag-and-Drop Branding: Upload logos and donation QR codes directly in the browser—saved locally to load instantly, even when offline.
  • Smart Typography Layouts: An auto-scaling host list engine that fits between 5 and 50 names perfectly, automatically adjusting padding and font sizes across centered, justified, or multi-column grids.
  • On-Screen Touch Correction: Tap and hold a name directly on the display to instantly remove it—perfect for making quick typo fixes or last-minute additions—with full undo history in the panel.
  • Live Aesthetic Calibrations: Quick-adjust sliders for background transparency, margin spacing, text sizes, custom color matching, and frame-rate limits.
Quick Controls Panel: Real-time FPS monitoring, wake lock, primary settings toggles.
Host Name Management: Live editing of host names with automatic duplicate filtering.
Branding and Content: Live text configurations, logo and QR code upload fields.
Appearance Customizer: Control particle speeds, counts, margins, backdrops, layouts, and more.

Event-Day Reliability

A digital kiosk must be rock solid on event day. Motion Poster is built to run continuously and autonomously, eliminating the stress of live tech glitches:

  • Screen Wake Lock: Automatically blocks screensavers and sleep modes, keeping the display perfectly active and bright for the entire event.
  • Instant State Recovery: Remembers all customized text and settings, automatically relaunching in full-screen presentation mode if a browser or kiosk power flicker occurs.
  • Fluid Multi-Device Scaling: Buttery-smooth graphics optimized to scale beautifully from mobile tablets to large 4K TV displays.
  • 100% Offline Resilience: The application runs fully client-side, ensuring the display never freezes or turns blank even if the venue loses internet connectivity entirely.
Technical Specifications
  • Language Core: Vanilla ES6+ Javascript, HTML5, CSS3
  • Rendering Pipeline: Dynamic HTML5 Canvas 2D & hardware-accelerated CSS transforms
  • Asset Persistence: LocalStorage using Base64 data-URI media serialization
  • System Integration: Screen Wake Lock API (with silent video loop fallback) and Fullscreen API
  • Device Targets: Pixel-perfect fluid scaling optimized for 1080p and 4K digital signage
